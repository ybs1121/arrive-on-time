추천 출발 시간 계산기

카카오 지도에서 출발지와 도착지를 검색해 직접 선택하고, 선택한 장소 이름과 좌표를 서버로 보내 추천 출발 시각을 계산합니다.

카카오 지도 JavaScript 키가 설정되지 않았습니다. `KAKAO_JAVASCRIPT_KEY` 환경변수를 설정하고, 카카오 개발자 콘솔에서 `http://localhost:8080` 도메인을 등록한 뒤 다시 실행해주세요.

장소 검색과 좌표 선택

장소를 검색한 뒤 목록에서 하나를 선택하면 해당 위치가 지도에 표시되고, 서버 요청에는 이름과 위도/경도가 함께 포함됩니다.

선택된 출발지
아직 선택되지 않았습니다.
선택된 도착지
아직 선택되지 않았습니다.

지도 미리보기

선택한 장소는 지도 위에 마커로 표시됩니다.

추천 요청

출발지와 도착지를 먼저 지도에서 선택한 다음, 도착 조건을 입력해서 추천 결과를 계산하세요.

도착 정책이 느슨할 때 최종 도착 기준 시각에 반영됩니다.
